Simple Problem (I'm Just A Kid)

Hi Non-Technical Friend! Appologies but the handbook made me call you that. I know you're technical cause otherwise how else did you get this link? I'm going to tell you about this time I got stuck on something as silly as pulling the plastic tag off of a drink bottle before you can drink.

I was writing some software for a friend using Django.
It did lots of clever math for property investing.
I was trying to loop through a tuple and it was not having a bar of it. I used to write this code between 10pm and 12am during the week so I wasn't the freshest.

I eventually solved it by using the python REPL and trying to loop through the data manually before pushing it.
I knew what I wanted. I told my dog what I was trying to do. He did not even offer a paw of support. Just the blank souless stare of a cavalier.

During the coding phase I felt annoyed but when I got there in the end I felt
-> great. Smarter, more aggressive. I felt like I could take on the world.

Elegant Solution

I recently had to code some JavaScript graphs for said friend. It was such a pain. Using Python to get values on the backend was easy. When I added the value into the graph it would just JavaScript fail(TM). You know... one little mistake and the entire JS file doesn't work.
If only there was a way to display bugs in the console.

My technique was to fill in the values in the JavaScript manually.
Get the Python code to push the values in the EXACT way JavaScript wanted it.
I wrote a function that took in the math and returned it formatted to work.
I knew I could do this and that I just had to get the write formatting.

A lot of the numbers had "currency" formats and the graph for some reason did not like ", or .00" which is what broke it.
The function took value, dropped the "," and ".00"
This allowed the graph to work and display the correct data.

I knew I could solve this during the coding which gave me the push to follow through.
Solving this made me feel great, especially seeing working graphs on an app.
Everyone knows that numbers aren't worth anything unless they are in a graph.
