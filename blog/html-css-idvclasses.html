◙◙◙ CSS: ID vs Classes ◙◙◙

CSS makes all the webthings pretty but like a pretty cat laying on it's back... tread lightly!

The first warning is: Consistency is King

Humans are amazing albeit flawed creatures. Sometimes we get lazy or cut corners and it might seem like it's the right thing to do at the time, it's never is. Consistency is where Class selectors come in.
You use Class selectors for groups or multiple items. Like you would make all the <li> elements the same with a class.

₪ Think: Class = Uniform ₪

IDs would be used if you wanted to do ONE thing. IDs are used to identify an element on your page for either styling or JavaScript magic.

Classes are selected with a . like so .hello-world {}
IDs are selected with a # like so #hello-world {}

For example if you inserted an image in your webpage:
<img src="img_gerard.jpg" alt="Gerard in a jacket" class="gallery">
